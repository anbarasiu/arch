## SEO Enhancements

This site includes comprehensive SEO optimization:

- **Meta Tags**: Enhanced title, description, and keywords for better search visibility
- **Open Graph**: Rich previews on Facebook, LinkedIn, and other social platforms
- **Twitter Cards**: Beautiful link previews when shared on Twitter
- **JSON-LD Structured Data**: Schema.org markup for Person and WebSite
- **Semantic HTML**: Proper heading hierarchy and ARIA labels
- **AI Crawler Permissions**: Explicit allow in robots.txt for GPTBot, Claude-Web, PerplexityBot, etc.
- **XML Sitemap**: Complete site structure for search engines
- **LLM-Friendly Content**: Hidden content section optimized for AI understanding

## Development

This is a static HTML site hosted on GitHub Pages.

### Local Testing

```bash
# Clone the repository
git clone https://github.com/anbarasiu/arch.git
cd arch

# Open in browser
open index.html
```

### Deployment

Push to `main` branch - GitHub Pages will automatically deploy.
